## 🎯 About

A sleek, modern, and responsive **developer portfolio** built with **Next.js 16 (App Router)**, **Tailwind CSS 4**, and **Framer Motion**. Designed with a premium dark theme, smooth animations, and component-based architecture to showcase projects, skills, and experience.

> 🚀 Built to impress. Designed to perform.

## 🚀 Getting Started

### Prerequisites

- **Node.js** ≥ 18.x
- **npm** or **yarn**

### Installation

```bash
# Clone the repository
git clone https://github.com/deepanshu83/my-portfolio.git

# Navigate to the project
cd my-portfolio

# Install dependencies
npm install

# Start the development server
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) to view it in your browser.

### Build for Production

```bash
npm run build
npm start
```

## 🎨 Customization

All portfolio data is centralized in the `data/` directory. Simply update these files to make it your own:

| File | What to Update |
|------|---------------|
| `data/profile.ts` | Name, bio, email, social links |
| `data/projects.ts` | Your projects, descriptions & links |
| `data/skills.ts` | Technical skills & soft skills |
| `data/experience.ts` | Experience, education & certifications |
